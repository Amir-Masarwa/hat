## **Technology Stack**

### **Frontend**
- **Framework:** React 18 with TypeScript
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios (with credentials)
- **State:** React Hooks (useState, useEffect, useMemo)
- **Build:** React Scripts (Create React App)

### **Backend**
- **Framework:** NestJS 10 with TypeScript
- **ORM:** Prisma 5
- **Authentication:** Passport JWT, bcryptjs
- **Validation:** class-validator, class-transformer
- **Testing:** Jest, Supertest

### **Database**
- **DBMS:** PostgreSQL 16 (Docker)
- **Connection:** Prisma Client
- **Migrations:** Prisma Migrate

### **DevOps**
- **Database:** Docker Compose
- **Development:** Hot reload (backend), Fast refresh (frontend)
- **Logs:** File-based (mailbox.log, ip-denied-attempts.log)
